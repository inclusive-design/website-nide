ATRC Self Voicing Chat

Requirements and Setup

To use the ATRC Self Voicing Chat, you need the following:

  • Microsoft's Internet Explorer or Netscape's Navigator
  • Sun's Java 1.1.X Plug-in
  • The IBM Self Voicing Kit (SVK)
  • ChatPerk.jar
  • Configure your system

Netscape's Navigator or Microsoft's Internet Explorer

The Chat is an applet that will run in a browser window. To use it, you must have a browser capable of running Java applets.

return to top

Sun's Java 1.1.X Plug-in

The IBM Self Voicing Kit requires Sun's Java Plug-in. You can obtain this from Sun's Java Plug-in page. Install the plug-in as instructed.

Note: The current version of the IBM Self Voicing Kit will not work with the most recent version of the Java Plug-in, due to a bug in the plug-in. Be sure that you obtain version 1.1.2 of the plug-in, not version 1.2.2.

return to top

The IBM Self Voicing Kit (SVK)

The kit can be downloaded from the Self Voicing Kit site. Install the SVK as instructed.

Note: The SVK is experimental software that has not been fully tested, and is made available by IBM for evaluation purposes only. For more information, see the License Agreement provided with the SVK.

Note: Due to a bug in the plug-in, you must use the SVK for JDK 1.1.x.

return to top

ChatPerk.jar

This is the Java archive that specifies how the Self Voicing Kit will voice the Chat. It must be downloaded and placed in your CLASSPATH. In some installations, the file must be place in the same directory as the svk.jar file.

Download the file ChatPerk.zip which contains ChatPerk.jar

return to top

Configure your system

Once all the necessary components have been installed on you system, you need only configure the SVK by editing the file ibmsvkconfig.properties, located in the lib directory off of your Java home directory.

Add the following line to the file:

Chat=atrcuoft.chat.ChatPerk
