Welcome to the NIDE Cascading Style Sheet (CSS) Generator

Cascading Style Sheets (CSS) control how web pages are displayed. Everything from font colors to image borders can be controlled using CSS. Usually, style sheets are used by web designers to enhance a web site. However, they can also be used by individual web users to override web sites' formatting. Users may wish to do this simply due to their own preferences, or because they have special needs not addressed by most web sites. For instance, a person with low vision browsing the web may wish to use a personal style sheet to increase the size of text within the web pages he or she visits.

Because Cascading Style Sheets are somewhat complicated to make and require knowledge of CSS code, this utility has been made. It will automatically generate style sheets for users without requiring any code knowledge whatsoever. Once you access the CSS Generator, you can either select a pre-configured style sheet that has been designed to meet special needs, or you can design a style sheet according to your own needs. The utility will then create the sheet and email it to you along with instructions on how to incorporate it into your web browser.
