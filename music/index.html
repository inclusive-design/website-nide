Music Notation Project



JScoreML is an on-line musical notation editor with playback and midi file creation capabilities. It was developed in association with the NIDE project. JScoreML was created entirely in the Java 2 programming language as an applet and uses an XML-based markup language to describe the score created by the user insofar as the rendering and playing of the score are implemented.
